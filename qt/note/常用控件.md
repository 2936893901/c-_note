# 常用控件
## 🔘按钮组
### `QPushButton`

### `QToolButton`
- 工具按钮
### `QRadioButton`
- 单选按钮
- 设置默认
```c++
setChecked(true);
```
### `QCheckButton`
- 多选按钮
- 监听状态(0:未选中,1:半选,2:选中)
## 列表容器`QListWidget`
```c++
QListWidget *listWidget = new QListWidget(parent);
QListWidgetItem *item = new QListWidgetItem("first");
listWidget->addItem(item);
```
